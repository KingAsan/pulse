# HDRezka Настройка Прокси

## Проблема
HDRezka.ag заблокирован в вашем регионе. Интеграция HdRezkaApi работает, но требует доступа к сайту.

## Решение

### Вариант 1: Использовать VPN (Рекомендуется)
1. Установите VPN (ProtonVPN, Windscribe, etc.)
2. Подключитесь к серверу в другой стране (США, Германия)
3. Запустите приложение - HDRezka будет работать напрямую

### Вариант 2: Настроить HTTP прокси
1. Найдите рабочий HTTP прокси
2. Добавьте в `backend/.env`:
   ```env
   HDREZKA_PROXY=http://your-proxy:port
   ```
3. Обновите `hdrezka_api_service.py` чтобы использовать этот прокси

### Вариант 3: Деплой на VPS
Задеплойте приложение на сервер в другой стране:
- Railway.app (США/Европа)
- Heroku
- DigitalOcean

На VPS HDRezka.ag будет доступен напрямую.
